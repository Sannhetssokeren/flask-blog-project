Проект: Flask-блог

01. Описание
Этот проект представляет собой простой веб-блог, созданный с использованием микрофреймворка Flask.
- Страница содержит информацию о достижениях и навыках автора.
- Используются HTML-теги и стилизация с помощью CSS.
- Включена таблица с прогрессом по дисциплинам.

02. Структура файлов
- `app.py`: Основной файл Flask-приложения.
- `templates/`: Директория для HTML-шаблонов.
  - `blog.html`: Шаблон главной страницы блога.
- `static/`: Директория для статических файлов.
  - `style.css`: Файл стилей.
  - `img/`: (Опционально) Директория для изображений.
- `README.md`: Описание проекта.

03. Запуск
Для запуска приложения необходим Python 3.x и установленная библиотека `Flask`.

Установка Flask:
```bash
pip install Flask
```
